# PDFConverter
Convert PDF RGB to CMYK in Linux.

# How to

Tested in Ubuntu20.04LTS 64bit.

Install GhostScript

edit pdf_converter.sh for inserting correct name of InputPDF file and OutputPDF file.

Then run

sh pdf_converter.sh

pdfimages -list outputPDF.PDF

to check RGM or CMYK
